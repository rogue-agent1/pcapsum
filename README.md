# pcapsum
Network connection summary: list, filter, group by process/port.
```bash
python pcapsum.py                    # All connections
python pcapsum.py --listen           # Listening ports
python pcapsum.py --process node     # Filter by process
python pcapsum.py --by-process       # Group by process
python pcapsum.py --by-port          # Group by port
```
## Zero dependencies. Python 3.6+.
